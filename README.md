# College Website Management System

This is a full-stack college website project built using Flask (Python) and MySQL.  
The main goal of this project was to create a simple and user-friendly web platform where users can register, log in, and view important information like news and events.

The project focuses on authentication, structured database design, and clean frontend-backend integration.
